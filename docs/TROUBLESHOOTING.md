# Fixing Problems

| Problem | Possible Fix |
|---------|--------------|
| Server says eula not agreed | Edit `server/eula.txt` and set `eula=true` |
| Java version error | Install Java 17 and make sure `java -version` shows it |
| Port already in use | Something else is using 25565; stop that app or change port |
| Friends can't join | Make sure they are on same Wi-Fi and you set `server-ip` |
| Python not found | Install Python 3.10+ and restart PowerShell |
